

# تحسين تصميم رحلة توثيق الهوية بالكامل

## الهدف
إعادة تصميم رحلة التوثيق (رفع المستند → السيلفي → الفيديو) بتجربة حديثة، سلسة، ومتجاوبة مع جميع الشاشات (خاصة سطح المكتب الكبير).

## المشاكل الحالية
1. بطاقة معاينة الصورة على سطح المكتب صغيرة جداً (لا تستغل المساحة)
2. الانتقال بين الخطوات جامد وبدون حركة سلسة
3. التصميم العام قديم — لا يعكس مستوى المنتج
4. لا يوجد مؤشر تقدم واضح (Stepper) بين الخطوات الثلاث
5. عناصر الكاميرا/الفيديو غير متمركزة بصرياً
6. أزرار الإجراءات (إعادة، تأكيد، تسجيل) متفرقة

## التصميم الجديد

### 1. Dialog Container
- توسيع العرض على سطح المكتب: `max-w-3xl` → `max-w-5xl` مع `lg:max-w-6xl`
- ارتفاع ثابت مع scroll داخلي للمحتوى فقط (الـ stepper والـ footer ثابتان)
- خلفية متدرجة خفيفة + حواف ناعمة `rounded-2xl` + ظل أعمق

### 2. Stepper علوي ثابت
شريط تقدم أفقي يعرض 3 خطوات:
```text
●━━━━━●━━━━━○
المستند  السيلفي  الفيديو
```
- الخطوة الحالية: لون primary + scale 110%
- المنتهية: علامة ✓ خضراء
- القادمة: رمادي باهت
- خط واصل بين الخطوات يمتلئ تدريجياً

### 3. خطوة المستند
- منطقة Drop Zone كبيرة (min-h-[400px] على الديسكتوب)
- أيقونة كبيرة + نص واضح + زر ثانوي
- بعد الرفع: معاينة بحجم كامل مع إمكانية الاستبدال

### 4. خطوة السيلفي
- Layout عمودين على الديسكتوب: كاميرا (يسار) + إرشادات (يمين)
- إطار دائري حول الوجه مع guide overlay
- بطاقة المعاينة بعد الالتقاط: `aspect-square` بحجم `max-w-md` (بدلاً من الصغير الحالي)

### 5. خطوة الفيديو
- نفس مبدأ العمودين
- مؤقت دائري كبير (5 ثوان) مع animation
- مؤشر تسجيل أحمر نابض
- بعد الانتهاء: مشغل فيديو بحجم كامل مع controls

### 6. Footer ثابت
- زر "السابق" (يسار) + زر "التالي/تأكيد" (يمين) بحجم كبير
- مؤشر loading واضح أثناء الرفع

### 7. حركات وانتقالات
- Framer-motion-like transitions بين الخطوات (`fade + slide`)
- Skeleton أثناء التحميل
- Toast errors تبقى كما هي (تعرض الخطأ الحقيقي)

## ملاحظات حرجة
- **لا تعديل** على منطق رفع الملفات أو MIME types (مشكلة CRM bucket منفصلة)
- **لا تعديل** على API contracts أو portalInvoke
- جميع النصوص عبر `t()` — احترام نظام 12 لغة
- RTL/LTR صحيح في الـ stepper والـ layouts
- اختبار على viewport 1713px (المستخدم الحالي) + موبايل

## المخرج المتوقع
- بطاقات معاينة كبيرة مريحة على الديسكتوب
- Stepper يوضح الموقع في الرحلة
- انتقالات سلسة بين الخطوات
- شعور حديث متسق مع باقي البورتال
